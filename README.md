# kina
轻量级ES网关查询工具，界面和交互仿照 Kibana 控制台

#### 使用方法

插件主要分为左中右三个操作模块

左侧可输入网关地址和请求头信息，灵活配置

中间可输入自定义 DSL 语句，支持多个 JSON 体，提供「运行」、「复制」、「格式化」三个基本操作

右侧输出 ES 网关访问结果，以 JSON 形式展示

更多实现细节，请查看 [Kina浏览器插件](https://jyncode.com/show/217.html "Kina浏览器插件") 博客文章
